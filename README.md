# 🧮 Math For Minutes

Kids solve math problems to earn internet time. Parents control the settings via a PIN-protected panel.

## Features
- 6 difficulty levels: Easy → Percentages
- Earn minutes for correct answers, lose minutes for wrong ones
- PIN-protected parent settings
- SMS report to parent when session ends
- Countdown timer for earned internet time

## Getting Started

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
# Upload the /dist folder to any static host (Netlify, Vercel, GitHub Pages, etc.)
```

## Default Parent PIN
`1234` — change it in Parent Settings after first launch.
